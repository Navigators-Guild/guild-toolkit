# guild-portfolio

**Difficulty: Intermediate**

Generate a portfolio site from your project repos — a static site generator scoped to one purpose.

## Scope

- Read project registry from `~/.guild/data/projects.toml`
- Parse README.md from each project repo
- Generate static HTML pages with project descriptions and metadata
- Output a self-contained site directory

## Key Skills

- Markdown parsing (pulldown-cmark or similar)
- HTML generation and templating
- File I/O across multiple repos
- Static site generation patterns
