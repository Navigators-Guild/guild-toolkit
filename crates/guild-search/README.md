# guild-search

**Difficulty: Intermediate**

Search the guild knowledge base, curriculum docs, and common patterns.

## Scope

- Search crosslink knowledge hub via `guild_core::crosslink`
- Search local curriculum docs by keyword
- Fuzzy matching for approximate queries
- Ranked results with context snippets

## Key Skills

- Crosslink CLI integration
- Text search and fuzzy matching
- Result ranking and display formatting
