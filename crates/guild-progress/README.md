# guild-progress

**Difficulty: Beginner**

Track your curriculum progress — where you are, what you've completed, what's next.

## Scope

- Read curriculum checkpoints from `~/.guild/data/progress.toml`
- Display progress as a checklist with completion status
- Show next uncompleted checkpoint
- Mark checkpoints as complete

## Key Skills

- TOML file parsing via guild-core data model
- Formatted terminal output (tables, colors)
- CLI argument parsing with clap
