# guild-scaffold

**Difficulty: Beginner**

Set up a new project with the right structure: design doc template, crosslink init, gitignore, README skeleton.

## Scope

- Create a project directory from a template
- Initialize git repo and crosslink
- Generate `.gitignore`, `README.md`, `CLAUDE.md` skeleton
- Create `.design/` directory with a blank design doc template

## Key Skills

- File I/O and directory creation
- String templating
- CLI argument parsing with clap
