# guild-stats

**Difficulty: Intermediate**

Project health dashboard — test coverage, issue velocity, commit frequency, lines changed.

## Scope

- Parse git log for commit frequency and lines changed
- Run `cargo test` and parse output for test counts
- Query crosslink for issue velocity (opened/closed over time)
- Display a summary dashboard in the terminal

## Key Skills

- Git log parsing
- Process spawning and output capture
- Data aggregation and summary statistics
- Terminal dashboard formatting
