# guild-review

**Difficulty: Intermediate**

Submit a project for adversarial review, pull down feedback, and track review rounds.

## Scope

- Submit current project for review via crosslink
- Check review status and pull feedback
- Track multiple review rounds per project
- Update `~/.guild/data/reviews.toml` via guild-core

## Key Skills

- Crosslink CLI integration
- State management (review rounds)
- Formatted terminal output
